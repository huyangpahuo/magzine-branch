# hexo-theme-magzine-next

大家好，这个项目是我基于原作者的 [hexo-theme-magzine](https://github.com/forever218/hexo-theme-magzine) 修改和发展出来的一个小版本 !!!

你们可以把这个当做magzine不一样的皮肤,因为我会继承magzine的所有功能和大部分样式

因为我会使用大量的AI辅助,至于本主题稳定性和兼容性嘛,三体人懂我(=￣ω￣=),我只在乎能不能实现,纯黑盒

所以不建议大家现在使用我这个版本,不过试一下也可以

## 使用说明

使用方法和 [hexo-theme-magzine](https://github.com/forever218/hexo-theme-magzine) 一模一样,大家去看看吧

打开你 Hexo 根目录下的 `_config.yml`，把主题改成：

```
theme: magzine-next
```

安装依赖

```
npm install
```
